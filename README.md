# 學生讀書任務管理器

這是一個 Python 程式設計期末小組作業，使用 Tkinter 製作桌面介面，並用 JSON 檔案保存讀書任務資料。

## 小組成員

- 羅宇航
- 張雲瀚
- 陽明荏

## 主要功能

- 新增、編輯、刪除讀書任務
- 切換任務完成狀態
- 依任務名稱、科目、備註搜尋
- 依建立時間、截止日期、優先順序、完成狀態排序
- 顯示總任務數、已完成數、未完成數、逾期數與完成率
- 使用 `tasks.json` 自動保存資料

## 執行方式

```bash
python3 app.py
```

程式只使用 Python 標準函式庫，不需要另外安裝套件。

## 任務資料格式

```json
[
  {
    "id": 1,
    "title": "完成 Python 期末報告",
    "subject": "程式設計",
    "due_date": "2026-06-10",
    "priority": "高",
    "status": "未完成",
    "note": "先完成 GUI，再整理報告",
    "created_at": "2026-05-24"
  }
]
```
